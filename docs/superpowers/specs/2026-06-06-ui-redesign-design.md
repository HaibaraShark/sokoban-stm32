# UI 全面翻新设计文档

## 概述

推箱子游戏 320×240 横屏 UI 全面翻新。混搭两种风格：
- **游戏瓦片** → 质感风（立体、纹理、光源感）
- **信息栏/菜单/选关** → 现代扁平风（克制配色、卡片布局、圆角）

## 配色体系

```
底色 (深炭灰):   0x1082
卡片底 (深):     0x18E3
卡片底 (亮):     0x2965
分隔线:          0x3186
文字主色:        0xFFFF (白)
文字标签:        0xA514 (浅灰)
文字提示:        0x630C (暗灰)
高亮/强调:       0xFFE0 (黄)
金色 (通关):     0xFD20
绿色 (完成):     0x07E0
红色 (目标):     0xF800
```

## 各页面改动

### 1. 游戏瓦片 (tile.c — Tile_Draw)

**墙体 (GROUND_WALL)**
- 底色：暖灰石材 `0xA514`
- 4×4 砖缝网格：横竖线用 `0x8C51`
- 边缘：左上高光 `0xC618` + 右下阴影 `0x738E`（2px 宽）
- 拐角加深

**地板 (GROUND_FLOOR)**
- 底色：暖米色 `0xD69A`
- 极淡网格纹理：`0xC618` 细线
- 边框保留灰色

**目标点 (GROUND_TARGET)**
- 红色菱形保持
- 外层加浅红细圈 `0xF9AE`（半径+3），模拟发光

**箱子 (ENT_BOX)**
- 主体：暖橙 `0xFD20`
- 左上高光条（2px）：`0xFE80`（模拟光源左上）
- 右下阴影条（2px）：`0xB420`
- 深棕边框 `0x8A20`
- 中心保留淡十字纹 `0xD500`

**箱子归位 (ENT_BOX + GROUND_TARGET)**
- 立体箱子照常绘制
- 外框加双圈绿色 `0x07E0`（分别偏移 1px 和 2px）

**玩家 (ENT_PLAYER)**
- 头：加大半径（ts/4），蓝色 `0x001F`，加亮蓝高光点 `0x041F`（半径 ~2px 左上）
- 身体：梯形（上窄下宽），深蓝 `0x0018`
- 脚：两个灰色小方块 `0x8410`

| 顶栏 | 0-23 | 深黑底 `0x0000` + "SOKOBAN" 白字 |
| 分隔线 | 24 | 细线 `0x3186` |
| 关卡号 | 30 | "Lv.01" 大号白色数字(16pt) |
| 关卡名 | 48 | 灰色小字(12pt) |
| 分隔线 | 62 | |
| Steps 标签 | 68 | "Steps" 灰色(12pt) |
| Steps 数值 | 82 | 白色大字(16pt) |
| Best 标签 | 102 | "Best" 灰色(12pt) |
| Best 数值 | 116 | 白色数字(16pt) |
| Remaining 标签 | 136 | "Remaining" 灰色(12pt) |
| Remaining 数值 | 150 | "◆ 2/4" (红色菱形图标+白字) |

| 操作按钮 | 172-210 | UNDO / MENU 小按钮 |
| Key hints | 215-240 | 最小号灰色提示(8pt) |

**底色**：`0x1082`（深炭灰），非深蓝。

**按钮**：圆角小按钮，深底 `0x2965` + 细白边框。

### 3. 主菜单 (menu.c — Menu_Draw)

**布局**
- 顶部居中：程序绘制的箱子小图标（12×12）
- 标题 "S O K O B A N"（字母间距 2px，32pt，青色）
- 装饰线 `───`
- 中文副标题 "推箱子"（16pt，白色）
- 3 个菜单按钮，间距 40px，起始 y=105

**按钮样式**
- 尺寸：180×36
- 底色：`0x18E3`（深卡片）
- 圆角：4px（用短横线模拟四角切角）
- 选中态：左边加亮色竖条（4px 宽，黄色 `0xFFE0`）+ 文字变黄
- 未选中：白色文字，无左边条

**底部**："NUAA Embedded 2025"（12pt，暗灰 `0x630C`）

### 4. 选关页面 (ui.c — UI_DrawLevelGrid)

**布局保持** 4×4 网格不变，每个格子 60×50。

**卡片样式**
- 已通关：金色底色 `0xFD20`，白色数字，3 颗星星
- 已解锁未通关：深卡片色 `0x2965`，白色数字
- 未解锁：暗色 `0x1082`，画小锁图标（长方形+半圆，灰色），无数字
- 选中态：边框加粗+亮色（黄色双线）
- 圆角：4px

**锁图标**（未解锁关卡）
- 8×10 的小图案
- 锁体：灰色矩形（4×6）
- 锁梁：灰色半圆环（半径 3，宽 2px）

**星星**：黄色填充五角星（简化画法），灰色空心 = 未获得

### 5. 按钮通用样式 (ui.c — UI_DrawButton)

- 圆角 4px（四角各切 2px 通过不画角落像素）
- 选中：左边条 + 双细线边框
- 未选中：单细线边框 `0x4208`
- 底色：`0x18E3`
- 文字居中

### 6. 其他页面微调

**帮助页面**
- 保持现有结构，改用新配色底色 `0x1082`
- 分区标题加下划线分隔

**启动画面**
- 保持动画流程不变
- 标题颜色从纯青改为渐变更丰富的青色 `0x04FF`
- 进度条加圆角端点

**胜利动画**
- 保持现有阶段结构
- "LEVEL CLEAR!" 加大到 32pt
- 星星从圆圈改为填充五角星

## 约束

- C89 语法，ARMCC V5 编译器
- 无位图资源，全部程序化绘制
- 320×240 横屏，ILI9341 LCD
- 中文用 GB2312 hex 转义
- 帧率敏感操作（脏矩形重绘）不额外增加耗时
